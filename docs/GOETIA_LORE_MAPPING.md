# GOETIA — mapping court pour la v2

Ce document résume le mapping de design actuellement utilisé dans le prototype. Le document source complet peut être enrichi ensuite avec les 72 démons.

## Principe

Dans GOETIA, le joueur ne fabrique pas des zombies génériques. Il contraint des **aspects démoniaques** dans des corps. Les démons sont des **sceaux de production**.

| Lore Ars Goetia | Mécanique GOETIA |
|---|---|
| Démon lié dans un sigil | Sceau de production |
| Légions commandées | Potentiel de horde |
| Rang | Tier |
| Domaine | Comportement |
| Rituel de contrainte | Coût en âme + corps |
| Sigil | Icône / recette |

## MVP demons

### Murmur — Soul Extractor

- Rôle : extraction d'âme.
- Usage : clic droit / E sur un cadavre.
- Design : tutoriel central, pouvoir de base.

### Bifrons — Hauler

- Rôle : transport de cadavres.
- Usage : serviteurs qui ramènent les corps à la fosse.
- Design : cœur logistique du jeu.

### Leraje — Archer

- Rôle : défense à distance.
- Usage : première unité de combat, protège la fosse et les porteurs.
- Design : transforme le stock âme/corps en protection active.

## Alpha demons

### Andras — Berserker instable

- Rôle : combat brutal.
- Twist : friendly fire ou désobéissance si stabilité rituelle basse.

### Malthus — Builder

- Rôle : construction et réparation.
- Usage : tours d'os, réparation de fosse.

### Buer — Support

- Rôle : restauration.
- Usage : soigne la horde, restaure la stabilité.

## Règle de nommage

Éviter d'avoir littéralement “12 Bifrons” à l'écran.

Préférer :

- Porteur de Bifrons
- Archer de Leraje
- Brute d'Andras
- Exarque de Murmur
- Gardien de Sabnock

Les démons restent des puissances supérieures. Les unités sont des corps marqués par leur sceau.
